Ayuda - Heatmap de Precio por m²
¿Qué es un heatmap?

Un heatmap (mapa de calor) es una representación visual de datos donde los valores se representan mediante colores. En este caso, muestra la densidad de precio por metro cuadrado en diferentes zonas geográficas.

¿Cómo interpretar los colores?
  • Verde: Zonas con precio por m² bajo (más económicas)
  • Amarillo: Zonas con precio por m² medio
  • Rojo: Zonas con precio por m² alto (más exclusivas)
Funcionalidades disponibles
  • Filtros: Puedes filtrar por tipo de propiedad, rango de precio, área y fuente de datos
  • Controles de capa: Activa/desactiva el heatmap o los marcadores individuales
  • Opacidad: Ajusta la transparencia del heatmap para ver mejor el mapa base
  • Propiedades destacadas: Lista de propiedades con mayor y menor precio por m²
Fuentes de datos
  • Propiedades locales: Base de datos interna del sistema
  • Propiedades Propifai: Base de datos externa de Propifai
