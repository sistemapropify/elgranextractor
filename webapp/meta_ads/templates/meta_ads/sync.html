Comando Manual

Para sincronizar datos manualmente desde la terminal, ejecuta:

python manage.py sync_meta_ads

Opciones adicionales:

python manage.py sync_meta_ads --days 60 Sincroniza insights de los últimos 60 días python manage.py sync_meta_ads --verbose Muestra detalles de la sincronización python manage.py sync_meta_ads --force Fuerza resincronización completa
