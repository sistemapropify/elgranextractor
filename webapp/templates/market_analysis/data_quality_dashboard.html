Valores Atípicos - Propify
Detección de Anomalías

Los valores atípicos se detectan usando el método IQR (Rango Intercuartílico). Valores fuera de 1.5 veces el IQR se consideran anomalías.

Nota: Algunos valores atípicos pueden ser legítimos (propiedades de lujo). Revise manualmente antes de eliminarlos.
Posibles Duplicados - Base Local
Posibles Duplicados - Propify
Gestión de Duplicados

Se detectan posibles duplicados basados en coordenadas, URLs y descripciones similares. Los umbrales de alerta ayudan a priorizar la revisión.

Importante: Los duplicados detectados automáticamente requieren validación manual. Algunos pueden ser propiedades similares pero distintas.

Campos Vacíos - Propify
Estrategia para Campos Vacíos
  • Prioridad 1: Campos críticos (precio, coordenadas) deben tener menos del 10% de vacíos
  • Prioridad 2: Campos importantes (área, ubicación) deben tener menos del 20% de vacíos
  • Acción: Implementar validaciones obligatorias en formularios de ingesta

Cómo interpretar este dashboard

Puntaje de calidad: Calculado combinando completitud (70%), duplicados (20%) y valores atípicos (10%). Un puntaje ≥80 indica excelente calidad, ≤40 requiere atención inmediata.

Actualización: Los datos se actualizan automáticamente cada 5 minutos. Use el botón "Actualizar" para forzar una actualización inmediata.
