Instrucciones
  1. Sube un archivo Excel o CSV con los datos de propiedades.
  2. Valida el mapeo de columnas y crea los campos necesarios.
  3. Usa Procesamiento con IA para extraer campos dinámicos de descripciones textuales.
  4. Procesa el archivo para importar los datos a la base de datos.
  5. Revisa los resultados y verifica que los datos se hayan importado correctamente.
