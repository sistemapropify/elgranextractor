Propiedades Propify - Página de Acceso

¡Importante! Esta página proporciona enlaces directos a las propiedades de la base de datos Propify. Hay 43 propiedades disponibles en la base de datos.

Vistas Disponibles

1. Vista Directa (HTML Generado)

Muestra todas las propiedades Propify en HTML simple generado directamente por Django.

Ventajas: No requiere recarga del servidor, muestra todas las propiedades inmediatamente.

Abrir Vista Directa
URL: http://127.0.0.1:8000/ingestas/propiedades-propify-directa/

2. Vista Temporal Propify

Vista que usa el sistema de templates de Django para mostrar propiedades Propify.

Abrir Vista Temporal
URL: http://127.0.0.1:8000/ingestas/propiedades-propify/

3. Vista Principal con Filtro Propify

La vista principal de propiedades con el filtro "Propify" activado.

Muestra propiedades Propify intercaladas con otras propiedades.

Abrir con Filtro Propify
URL: http://127.0.0.1:8000/ingestas/propiedades/?fuente_propify=propify

4. Vista Principal Completa

Todas las propiedades (locales, externas y Propify) en la vista principal.

Las propiedades Propify aparecen con un badge verde "Propify".

Abrir Vista Principal
URL: http://127.0.0.1:8000/ingestas/propiedades/

Información Técnica

  • Base de datos Propify: SQL Server con tabla "properties"
  • Propiedades encontradas: 43 propiedades
  • Con coordenadas: 42 propiedades tienen coordenadas válidas
  • Modelo Django: PropifaiProperty en webapp/propifai/models.py
  • Configuración: Base de datos de solo lectura (managed=False)
Nota sobre el servidor: El servidor está ejecutándose con --noreload, por lo que los cambios en archivos Python no se recargan automáticamente. La vista directa debería funcionar inmediatamente.

Solución de Problemas

Si no ves propiedades Propify:

  1. Prueba la Vista Directa primero (debería funcionar siempre)
  2. Verifica que el servidor esté ejecutándose: http://127.0.0.1:8000/
  3. Ejecuta el script de verificación: python verificar_propify_directo.py
  4. Revisa los logs del servidor para errores
